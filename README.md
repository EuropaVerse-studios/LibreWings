# LibreWings

**LibreWings** is an open-source aviation game developed in Godot 4.5 by a single passionate student who started the project and made it free and open source for everyone.
Fly fighters and bombers, complete adrenaline-filled missions, mod your game, feel like a real pilot but from your house and contribute to an ever-evolving open-source project!

## Features
- Realistic yet fun flights
- Combat systems: missiles, cannons, radars...
- Simplified HUD and cockpit
- Training missions and dogfights
- Open source: everyone can contribute!
  
## Getting Started
### From GitHub
1. Download or clone the repository
2. Open `project.godot` with Godot 4.5
3. Launch the game and test the prototype
### From Itch.io
1. Click the link in the Useful links section
2. Scroll down the page and find the download section
3. Dowload the game file for your OS
4. Enjoy the game!

## How to Contribute
- Check the [docs/contributing](docs/contributing) folder, there you can find manuals both in english and italian
- Follow the roadmap [docs/roadmap.md](docs/roadmap.md)
- Check the [docs/changelog.md](docs/changelog.md) to see what was added in which version
- File issues for bugs or new ideas
- Submit a Pull Request when adding code, assets, or missions
