# Contributing to LibreWings
Thank you for your interest in LibreWings!
LibreWings is an open source project developed with Godot 4.5, designed to grow over time thanks to community collaboration.
Whether you are:
-programmer
-3D artist
-sound designer
-tester
-or simply full of ideas

you are welcome.
This guide explains how to contribute correctly, step by step.

## Project philosophy
LibreWings aims to be:
-open source
-educational
-fun
-collaborative
-technically clean

We are not looking for absolute perfection, but for:
clear code, sensible ideas, and concrete improvements

## Important rules
License:

LibreWings uses GPLv3.
By contributing you agree that:
-your code remains open source
-your changes stay under GPLv3

What is NOT accepted:
-code copied from closed source projects
-stolen or non-original assets
-cheats, malware, backdoors
-toxic or aggressive behavior

Code of conduct:
Be:
-respectful
-collaborative
-constructive

We don‚Äôt need arrogant geniuses.
